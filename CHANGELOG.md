# Changelog

## 2026-05-04

### Added

- Created the first School Helper web prototype.
- Added a conversational AI teacher interface.
- Added school material upload support for readable text files.
- Added pasted-note support for classroom notes and study material.
- Added local material summarization and keyword extraction.
- Added four learning modes: explain, quiz, review, and homework.
- Added quick prompt buttons for common learning tasks.
- Added source chips so answers can show which uploaded materials were used.
- Added responsive layout for desktop and mobile screens.
