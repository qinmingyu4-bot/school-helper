# School Helper

一个对话型 AI 教学助手网页原型。学生可以上传学校资料或粘贴课堂笔记，网页会用“AI 老师”的方式进行讲解、答疑、出题和复习辅助。

## 功能

- 上传 TXT、MD、CSV、JSON、HTML 等文字资料
- 粘贴课堂笔记并加入资料库
- 自动整理资料重点
- 支持讲解、测验、复习、作业四种学习模式
- 基于上传资料生成答疑内容和引用来源
- 纯前端运行，不需要安装后端

## 使用方式

直接打开 `index.html` 即可使用。

如果部署到 GitHub Pages，可以在仓库设置里开启 Pages，并选择 `main` 分支根目录。

## 文件结构

```text
index.html  页面结构
style.css   页面样式
app.js      资料读取与对话逻辑
```
